Find trivia nights with an interactive map and filterable list.

== Description ==

Trivia Night Finder displays trivia events on an interactive Google Map with a filterable list view.

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/trivia-night-finder`
2. Activate the plugin through the 'Plugins' menu
3. Add the shortcode `[trivia_finder]` to any page or post

== Shortcode Usage ==

Basic usage:
`[trivia_finder]`

With custom title:
`[trivia_finder title="Find Trivia Events"]`

With custom CSV URL:
`[trivia_finder csv_url="https://your-csv-url.com/data.csv"]`
